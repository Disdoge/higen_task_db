# 何海军 Python 练习与测试开发作品集

这个仓库用于记录我的 Python 小程序练习、测试开发练习和个人主页。

## 在线主页

后续开启 GitHub Pages 后，可以通过网页访问个人主页。当前主页入口文件是：

```text
index.html
```

## 项目目录

- `01_calculator`：Python 基础计算器
- `04_pytest_api_demo`：Pytest 接口测试样例
- `05_bug_report_analyzer`：Bug 和日志文本分析样例
- `06_cli_todo_app`：命令行待办工具样例
- `07_portfolio_site_sample`：个人主页维护样例

## 日常流程

在 VS Code 中：

1. 运行任务 `New Python Project` 创建新项目
2. 写代码并在终端运行 `python 项目文件夹/main.py`
3. 运行任务 `Save and Push` 上传到 GitHub
